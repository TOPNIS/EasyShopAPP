# EasyShop - Aplicacion Movil Android 
 
Aplicacion de comercio electronico especializada en productos tecnologicos, desarrollada en **Kotlin** con **Jetpack Compose**. 
 
## 12 Requerimientos Implementados 
 
1. Splash Screen 
2. Catalogo de Productos 
3. Detalle de Producto 
4. Sistema de Favoritos 
5. Busqueda y Filtrado 
6. Comparador de Productos 
7. Historial 
8. Base de Datos Room 
9. Arquitectura MVVM 
10. Formularios Validados 
11. Recursos Nativos (Camara y Galeria) 
12. Animaciones
